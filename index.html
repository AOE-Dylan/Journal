UX and UI

8-23-16
Difference Between UX and UI

UX stands for user experience. This encompasses the styling of the webpage and how the user feels when using the program.

UI stands for User Interaction. This is everything the user interacts with in the program, like links and functions.

HTML and CSS

9-28-16
HTML

HTML or HyperTextMarkupLanguage, is the base code that makes up the content of the document.It specifies what elements are in the head and body. It provides the basic structure for a website.

CSS

CSS otherwise known as Cascading Style Sheets, is used to style html code. It provides the backround, colors, fonts, spacing, and many other things including margins and padding.

Responsive Design

9-28-16
Responsive Design

Responsive design is web-based design where the content is flexible and fluid and adapts to the size of the screen.

In responsive design, you would normally use adaptable units such as em and percentages. Also you would use CSS media queries to specify what happens to certain elements when the screen resolution and etc. Is changed.

CSS Media Queries

10-7-16
CSS Media Queries

CSS Media Queries are used in responsive web design. They help to fit the content to the screen whenever you change the size and/or resolution. They are an important part of grid based design because elements need to consistently flow on the webpage. An example of a media query would be: @media only screen and (min-width: 500px). Under this you would put whatever special styles would be used for a screen that is larger than 500px.
